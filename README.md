# RoadRunnerGame
A 3D obstacle course game built in Unity where players must navigate through a series of platforming challenges using movement mechanics such as running, jumping, and double-jumping. The objective is simple: avoid obstacles, stay on the course, and reach the finish line as quickly as possible.

This project was created to strengthen my understanding of Unity game development, C# scripting, player movement systems, physics interactions, collision detection, UI development, and game design fundamentals.


## Gameplay
The player progresses through a linear obstacle course filled with various challenges that test timing, movement, and precision.

Features include:

- Smooth player movement
- Jumping system
- Double-jump mechanic
- Physics-based obstacle interactions
- Respawn system when falling off the course
- In-game timer
- Finish line detection
- Victory screen upon completion
